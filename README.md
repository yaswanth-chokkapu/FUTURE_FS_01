# 💼 Yaswanth Chokkapu Portfolio

A premium, production-level developer portfolio built using React and Tailwind CSS, designed with a minimal and elegant Apple-inspired UI.

## 🚀 About

This portfolio showcases my work as a Computer Science Engineering student specializing in Artificial Intelligence. It highlights my projects, skills, and interest in building intelligent, scalable, and real-world applications.

## ✨ Features

* ⚡ Modern React (Vite) setup
* 🎨 Tailwind CSS (clean & responsive design)
* 🌙 Dark / Light mode toggle
* 🎬 Smooth animations using Framer Motion
* 📱 Fully responsive across all devices
* 📩 Functional contact form (email integration)
* 💎 Apple-inspired premium UI/UX
